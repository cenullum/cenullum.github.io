Pitch

cenker's mod is a multiplayer physics sandbox with many mods. Every mod has a different experience. Play included, community-made, and your own mods together with friends.

Description

cenker's mod creates a sandbox environment where players can explore a variety of community-created mods or host their own sessions with up to 32 players or more if it is a simple mod. From chaotic physics playgrounds to structured competitive games, the Steam Workshop integration ensures there is always something new to discover.

For those who want to create, the built-in Lua 5.4 API provides extensive capabilities. With an integrated Code Editor and Tile Editor, you can build intricate worlds and logic. Features like automatic network synchronization, dynamic UI creation, and 2D physics manipulation allow for complex interactions, all accessible directly within the platform.

Features

  • Multiplayer Support: Built-in Steam networking with host-client architecture
  • 3 Included Game Modes: Jump straight into action with three complete, ready-to-play mods: a competitive football game, Campfire Survivors (survive waves of enemies), and Hookup (race to the top with a grappling hook).
  • Steam Workshop: Discover and install new mods with a single click.
  • Sandboxed Security: Safe Lua environment ensures code runs securely on your machine.
  • Server Browser: Integrated lobby system with filters to find a lobby that is hosted by users around the world.
  • Entity System: Flexible entity-component architecture
  • Voice Chat: Integrated voice communication system

About Developer

I'm Cenker I use Cenullum as nickname, I have been making games since middle school. Even while studying Anesthesia in high school, I stayed committed to my hobby and went on to study Digital Game Design at university. Over the years, I have worked with engines like LibGDX, GameMaker, Unity, and Unreal.

cenker's mod is built with Godot, though I plan to use Defold for my future projects. It has been in development for 2 years, born from a desire to simplify multiplayer game development and bring people together through creativity and play.

The project started as a cross-platform prototype (Mobile, Web, PC) using AWS, but I eventually focused only on PC Steam. I realized I was rewriting the same networking code for every new game idea, so I decided to build a platform that handles that complexity once and for all.

The development took longer than I hoped. As a solo developer, I had to balance the project with various personal challenges that arose along the way. Despite these hurdles, my passion never faded, and I kept working on the game whenever I could.
